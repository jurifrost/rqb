Version 6 requires no code changes when migrating from version 5, but the behavior of the default [`ValueEditor`](./components/valueeditor) component when the `operator` is "between" or "notBetween" is different.

Instructions on migrating from v4 to v5 are [here](/docs/5/migrate).
